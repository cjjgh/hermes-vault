# Hermes Vault 🧠

Hermes AI Agent 的知识库与记忆系统，基于 Obsidian + Git 双同步。

## 目录说明

| 目录 | 用途 | 关联 Hermes |
|------|------|------------|
| `知识库/` | 长期知识沉淀（AI/编程/金融） | Hermes 推理时参考 |
| `进化训练/` | 自主进化报告与能力趋势 | ↔ `training_data/` |
| `Hermes记忆/` | 持久记忆与行为规则 | ↔ `MEMORY.md` / `AGENTS.md` |
| `技能工坊/` | 技能文档与待开发需求 | ↔ `skills/` |
| `日常笔记/` | 日常想法与学习记录 | 人用为主 |
| `模板/` | Obsidian 模板文件 | 新建笔记模板 |

## Git 同步

- 自动推送：每 10 分钟（Obsidian Git 插件）
- 手动：`git push`
