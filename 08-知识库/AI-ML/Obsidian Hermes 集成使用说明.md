# Obsidian + Hermes 知识库集成使用说明

> 本文档写于 Obsidian 中，通过反向同步自动喂给 Hermes。

## 什么是这个知识库

这是 Hermes 的辅助知识来源。你在 Obsidian 的 `08-知识库/` 目录下写的任何 Markdown 文件，每 30 分钟会被自动同步到 `~/.hermes/knowledge/`。Hermes 在会话中会主动读取这些内容。

## 怎么用

1. 在 Obsidian 里打开 `08-知识库/` 下的目录
2. 新建或编辑一个 Markdown 文件
3. 等待最多 30 分钟，或手动运行反向同步：

```bash
bash ~/.hermes/scripts/sync_vault.sh
```

4. 新对话中 Hermes 会自动知道这些知识

## 目录结构

| 目录 | 用途 |
|------|------|
| `08-知识库/AI-ML/` | AI/机器学习/LLM 相关知识 |
| `08-知识库/编程开发/` | 编程技术、最佳实践 |
| `08-知识库/金融分析/` | 金融、交易、市场分析 |
| `10-日常笔记/想法/` | 灵感、点子 |
| `10-日常笔记/学习记录/` | 学习笔记、教程 |

## 注意事项

- 文件格式：Markdown，中文或英文均可
- 文件名建议用英文或中英混合，避免纯中文文件名编码问题
- 内容不限，Hermes 会全文读取
