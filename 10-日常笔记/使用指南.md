# 📖 日常使用指南

> 如何高效使用 Hermes Vault + Obsidian

## 🔹 日常流程

### 看什么（Read）

- **最新进化报告** → `06-进化训练/最新报告.md`
- **能力趋势** → `06-进化训练/能力趋势/`
- **长期记忆** → `03-长期记忆/长期记忆.md`（Hermes 记住了什么）
- **蒸馏日志** → `05-蒸馏日志/KAIROS.md`（每日干了什么）

### 写什么（Write）

| 场景 | 写在哪 | 说明 |
|------|--------|------|
| 突然想到的点子 | `10-日常笔记/想法/` | 自由写，Hermes 同步时能看到 |
| 学习笔记 | `10-日常笔记/学习记录/` | 系统化记录 |
| 给 Hermes 的知识 | `08-知识库/AI-ML/` 等 | Hermes 会读到这些上下文 |
| 指导原则 | `02-行为规则/` | 想给 Hermes 加规则时写这 |

### 查什么（Search）

在 Obsidian 里直接按 `Cmd+P` 搜全文——比在飞书上翻聊天记录快得多。

## 🔹 与 Hermes 的互动

### 当你告诉 Hermes 事情时

Hermes 会自动把关键信息写入 `~/.hermes/memories/`，然后每30分钟同步到 vault 的对应目录。你可以随时在 Obsidian 里查看 Hermes 记住了什么。

### 当你在 vault 里写笔记时

目前是**单向同步**（Hermes → vault）。如果你想 vault → Hermes，可以：
1. 在 Obsidian 里写笔记
2. 告诉 Hermes："去看 vault 里 XXXX 目录下的内容"
3. Hermes 会读取对应文件

## 🔹 Obsidian 操作

### 打开 vault
```bash
open -a Obsidian ~/hermes-vault
```

### 快捷键
- `Cmd+P` — 命令面板（搜文件/功能）
- `Cmd+O` — 快速打开文件
- `Cmd+Shift+F` — 全文搜索
- `Cmd+N` — 新建笔记

### 推荐插件（安装后打开设置 → 社区插件）
1. **Obsidian Git** — 自动 git push/pull（设置自动同步间隔10分钟）
2. **Graph View** — 可视化知识关联

## 🔹 Git 同步

vault 自动每30分钟同步 Hermes 记忆到 GitHub。你也可以手动操作：

```bash
cd ~/hermes-vault
git add -A
git commit -m "手动更新"
git push
```

## 🔧 维护：当 Hermes 新增路径时怎么办

随着 Hermes 不断进化，可能会增加新的文件或目录。要让它们自动同步到 vault，只需一步：

### 操作步骤

```bash
# 编辑路径映射配置文件
vim ~/.hermes/scripts/sync_vault_config.sh
```

找到文件末尾的 **"未来新增路径请加在这里"** 区域，按格式加一行：

```bash
# 格式：   "源路径"                      "目标路径"
# 示例1：  "config.yaml"                "09-系统管理/配置文件/config.yaml"
# 示例2：  "training_data/reasoning"    "06-进化训练/训练数据/reasoning"
# 示例3：  "backups"                    "09-系统管理/备份"
```

保存后，下次 cron 自动同步（最多30分钟）就会生效。也可以手动立即跑一次：

```bash
bash ~/.hermes/scripts/sync_vault.sh
```

### 路径规则

| 前缀 | 含义 | 示例 |
|------|------|------|
| 无前缀 | 相对于 `~/.hermes/` | `"SOUL.md"` → `~/.hermes/SOUL.md` |
| `MEM:` | 相对于 `~/.hermes/memories/` | `"MEM:KAIROS.md"` → `~/.hermes/memories/KAIROS.md` |
| `ABS:` | 绝对路径（极少使用） | `"ABS:/tmp/report.md"` |

### 整体架构

```
sync_vault_config.sh     ← 你编辑这里（只改配置）
        │
        ▼
sync_vault.sh (v3)       ← 自动读取配置，执行同步（不用动）
        │
        ▼
~/hermes-vault/          ← Obsidian 知识库
```
